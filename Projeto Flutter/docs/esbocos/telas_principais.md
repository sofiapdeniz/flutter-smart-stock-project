# Esboços das Telas Principais - SmartStock Mobile

## 1. Tela de Login
Tela inicial com logo do SmartStock, campos de email e senha, e botão de entrar. Design minimalista com foco na autenticação.

## 2. Tela Principal (Dashboard)
Tela com menu lateral ou superior, exibindo 4 cards principais: Produtos, Fornecedores, Pedidos de Compra e Pedidos de Venda. Cada card leva para sua respectiva funcionalidade.

## 3. Tela de Listagem de Produtos
Lista de produtos cadastrados com campo de busca no topo. Cada item mostra nome, código, preço e quantidade. Botão flutuante (+) para adicionar novo produto.

## 4. Tela de Cadastro/Edição de Produto
Formulário com campos: Nome, Código, Descrição, Preço Unitário e Unidade de Medida. Botão de salvar na parte inferior.

## 5. Tela de Listagem de Fornecedores
Lista de fornecedores com nome, CNPJ e telefone. Campo de busca e botão para adicionar novo fornecedor.

## 6. Tela de Novo Pedido de Venda
Formulário com dados do cliente (nome, telefone, endereço), seleção de produtos, tipo de entrega (entrega ou retirada) e cálculo automático do valor total.

## Fluxo de Navegação
Login → Dashboard → [Produtos | Fornecedores | Pedidos Compra | Pedidos Venda]

## Padrão Visual
- Material Design
- Cores: Azul e branco
- Ícones do Material Icons
- Feedback visual para ações
